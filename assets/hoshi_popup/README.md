# Hoshi Reader popup renderer

`popup.js`, `popup.css`, and `selection.js` are vendored from
[Manhhao/Hoshi-Reader](https://github.com/Manhhao/Hoshi-Reader) and retain
their upstream copyright and SPDX notices. They are distributed under
GPL-3.0-or-later; see `LICENSE` in this directory.

Mangayomi supplies a Flutter JavaScript bridge in
`lib/modules/mining/widgets/hoshi_dictionary_popup.dart` for lookup entries,
dictionary media, nested lookups, external links, and Anki mining.
